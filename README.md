# HK Jewelry Corp - Sitio Web

Sitio web profesional para HK Jewelry Corp, una joyería de lujo ubicada en Miami, FL.

## Estructura del Proyecto

### Páginas HTML
- **index.html** - Página de inicio con hero section, colecciones destacadas, información sobre la empresa y productos destacados
- **products.html** - Catálogo de productos con categorías (collares, anillos, pulseras, aretes, conjuntos)
- **custom-jewelry.html** - Página de joyería personalizada con proceso de diseño y galería
- **luxury-watches.html** - Página de relojes de lujo con listados y garantía de autenticidad
- **about.html** - Página sobre nosotros con historia, valores y equipo
- **contact.html** - Página de contacto con formulario, mapa de Google Maps e información de contacto
- **terms.html** - Términos y Condiciones
- **privacy.html** - Política de Privacidad

### Archivos de Estilo y Scripts
- **styles.css** - Estilos globales con diseño moderno y lujoso (negro, blanco, acentos dorados)
- **script.js** - JavaScript para funcionalidades interactivas (formularios, animaciones)

## Características

- ✅ Diseño responsive (mobile-friendly)
- ✅ Paleta de colores lujosa (negro, blanco, dorado)
- ✅ Navegación intuitiva con menú superior
- ✅ Footer con información de contacto y enlaces legales
- ✅ Formularios de contacto y newsletter
- ✅ Integración de Google Maps
- ✅ Animaciones suaves al hacer scroll
- ✅ Optimizado para SEO con meta tags

## Instalación en Hostinger

1. Suba todos los archivos al directorio raíz de su hosting (public_html o www)
2. Asegúrese de que index.html esté en la raíz
3. Configure el formulario de contacto para enviar emails (puede requerir PHP)
4. Reemplace los placeholders "introduzca informacion aqui" con información real
5. Actualice el mapa de Google Maps con la ubicación correcta si es necesario

## Placeholders a Reemplazar

Busque y reemplace los siguientes textos en todas las páginas:
- `introduzca informacion aqui` - Información específica del negocio
- `introduzca telefono aqui` - Número de teléfono real
- `introduzca correo aqui` - Email de contacto real
- `introduzca horario aqui` - Horario de atención
- `introduzca fecha aqui` - Fechas en documentos legales

## Personalización

### Colores
Los colores principales están definidos en `styles.css` como variables CSS:
- `--color-black`: #000000
- `--color-white`: #ffffff
- `--color-gold`: #d4af37

### Imágenes
Las imágenes actuales son placeholders de Unsplash. Reemplace con imágenes reales de productos y del negocio.

### Formulario de Contacto
El formulario actualmente muestra una alerta. Para funcionalidad completa, configure un script PHP o use un servicio de terceros para el envío de emails.

## Notas

- El sitio está diseñado como catálogo (no e-commerce completo)
- Los productos tienen botones "Consultar Ahora" en lugar de "Agregar al Carrito"
- Todos los textos están en español según las especificaciones
- El diseño es completamente responsive y se adapta a dispositivos móviles
