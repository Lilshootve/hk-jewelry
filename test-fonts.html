Si ves Georgia o Times New Roman, las fuentes personalizadas no se están cargando.

Instrucciones para probar:

  1. Abre este archivo en tu navegador (test-fonts.html)
  2. Presiona F12 para abrir las herramientas de desarrollador
  3. Ve a la pestaña "Network" (Red)
  4. Recarga la página (Ctrl+R o F5)
  5. Busca archivos .TTF o .ttf en la lista
  6. Si aparecen en rojo o con error 404, las rutas están incorrectas
  7. Si aparecen en verde (200), las fuentes se cargaron correctamente

Nota: Si estás abriendo el archivo directamente (file://), puede haber problemas de CORS. Es mejor usar un servidor local como Live Server o Python: python -m http.server 8000
